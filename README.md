# 笺字 (JianZi)
见字如面，于方寸笺上，起落字间。

「笺字」是一个基于 `Canvas` 引擎 开发的高级书信排版编辑器。在即时通讯泛滥的时代，我们试图通过技术手段，找回纸笔书写时的那份重量感与仪式感。它不仅是一个文本编辑器，更是一个探索中式排版美学与数字交互边界的实验。

我一直觉得，用信笺来进行交流是一个很优雅的事情。我们在短短不到两三百字的信息中，向对方问好，询问近况，分享一些生活中的趣事，虽然说在即时通讯中，我们可以很方便的发送很多信息，但是总觉得少了点什么。

我仔细想了想，是缺少的琢磨，我们可以很随意地给我们的朋友发送信息，但是很少会去认真的思考，发送的信息是否合适？是否会给对方带来不好的影响？虽然说邮电在欧美国家很常见，大家都习以为常了，甚至有点烦闷，但是我觉得这也不失为一种多元世界的交流方式。

当然，这是二十二岁的自己在广州旅游的时候的一个小想法，现在过去一个月，偶尔写点，也差不多接近尾声了。
